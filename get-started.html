Getting Started

How to include the Pwn framework in your website.

Setting Up Your Page

Follow these instructions to get the Pwn framework running properly inside your HTML page.

Specify the HTML 5 doctype

In order for your page to properly respond to screen sizes, you must use HTML 5. This begins by specifying the doctype as html. At the very top of your page, add the doctype above your html tag: 

<!DOCTYPE html>

Modify your html tag to include the language. 

<html lang="en">

Include the meta viewport tag at the top of your head section. 

<meta name="viewport" content="width=device-width, initial-scale=1">

Include the Pwn CSS

Paste this <link> element to your page's <head> section: 

<link rel="stylesheet" type="text/css" href="//cdn.brytecore.com/pwn/css/pwn.min.css">

Include the Pwn JavaScript

If you decide to use any of the Pwn JavaScript components, you need to add a <script> tag at the end of your page, just before your </body> tag.

The Pwn JavaScript library requires jQuery. To put jQuery on your site, add a <script> tag linking to the library, before the Pwn script link. 

<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
<script type="text/javascript" src="//cdn.brytecore.com/pwn/js/pwn.min.js"></script>

Other Meta Tags

We recommend you add charset and http-equiv tags for maximum compatibility across devices and browsers.

The IE=edge statement tells Internet Explorer to use the latest rendering engine, preventing compatibility mode issues. 

<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">

Putting It Together

The very top of your html page should look very similar to this snippet. 

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<link rel="stylesheet" type="text/css" href="//cdn.brytecore.com/pwn/css/pwn.min.css">
	<title>Your page title</title>
</head>
<body>

The bottom of your page should look similar to this snippet. 

	<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
	<script type="text/javascript" src="//cdn.brytecore.com/pwn/js/pwn.min.js"></script>
</body>
</html>
