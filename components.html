Javascript Components

Components to help manage your user interface actions and behaviors.

Installing components on your website

The Pwn Javascript library requires jQuery. To put jQuery on your site, add this <link> to your page, before the Pwn javascript link. 

<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>

Copy pwn.min.js into your website and add a <link> tag at the end of your page, just before your </body> tag: 

<script type="text/javascript" src="/path-on-your-server/pwn.min.js"></script>

Checkbox

Replace the default html checkbox with a styled version that matches the Pwn theme. 

<!-- Don't forget the checkbox class on your label element. -->
<label for="checkbox-demo" class="checkbox">
	<input type="checkbox" id="checkbox-demo"> Yes, I'd like to see this demo!
</label>

<script type="text/javascript">
	// Don't forget to wrap your code in the jQuery .ready() function (not shown).
	$( "#checkbox-demo" ).pwn_checkbox();
</script>

Radio Buttons

Replace the default html radio buttons with a styled version that matches the Pwn theme. 

<!-- Don't forget the RADIO class on your label elements. -->
<label class="radio"><input type="radio" name="radio-demo"> You can go with this</label>
<label class="radio"><input type="radio" name="radio-demo"> Or you can go with that</label>

<script type="text/javascript">
	// Don't forget to wrap your code in the jQuery .ready() function (not shown).
	$( "label.radio input" ).pwn_radio();
</script>

Select Box

Custom drop-down boxes that match the style of other pwn form elements. Simply insert a <select> element on your page and run .pwn_select() on the jQuery selector for the element. 

<label for="select-demo">
	Choose your favorite make of sports car:
</label>
<select id="select-demo">
	<option>Ferrari</option>
	<option>Lamborghini</option>
	<option>Maserati</option>
	<option>Porsche</option>
</select>

<script type="text/javascript">
	// Don't forget to wrap your code in the jQuery .ready() function (not shown).
	$( "#select-demo" ).pwn_select();
</script>
