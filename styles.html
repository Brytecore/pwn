Typography

Paragraphs and Body Text

The base font used in Pwn is lato, an open source font family. The font adds about 82 KB to your page weight. In a future version, we will provide downloadable libraries with and without the font.

Wrapping text in <p> tags ensure there is plenty of margin between the sections of text, greatly improving readability. The default paragraph font size is 16px, weight is 300 (normal) and the line-height is 1.6em. Each paragraph has a .2em top and a 1.8em bottom margin.

You know, I have one simple request. And that is to have sharks with frickin' laser beams attached to their heads! Now evidently, my cycloptic colleague informs me that that can't be done. Can you remind me what I pay you people for? Honestly, throw me a bone here. What do we have? 

<p>
	…
</p>

Hyperlinks

Hyperlinks are the same font size as surrounding text, but colored #27a9e0 with an underline hover style.

Stay awhile, and listen. 
<a href="http://en.wikipedia.org/wiki/Diablo_II">Stay awhile, and listen.</a>

Forms

All form elements must be inside an element with the class form. This can be an actual <form> element, or simply a <div>. 

<form class="form">
	<!-- Your form elements go here. -->
</form>

<!-- You can also use elements other than <form> -->
<div class="form">
	<!-- Your form elements go here. -->
</div>

Text Box

Using the Pwn text box is as simple as using a normal HTML <input type="text"> element. The <label> element is optional, but recommended. Add a placeholder attribute to suggest a value or format to the user. 

<label for="ex-form-text-1">Why did you disobey your program?</label>
<input id="ex-form-text-1" type="text" placeholder="...">

Textarea

Create a multi-line text box using the HTML <textarea> element. The <label> element is optional, but recommended. Add a placeholder attribute to suggest a value or format to the user. 

<label for="demo-textarea-1">West of House</label>
<textarea id="demo-textarea-1" placeholder="..."></textarea>

Dropdown Box

Pwn radio buttons replace normal <select> elements inside your form. The Pwn JavaScript library is required for this form element. For full syntax and options, see the JavaScript Components page. 

<select id="select-demo">
	<option>Is the glass...</option>
	<option>Half full</option>
	<option>Half empty</option>
	…
</select>

<script type="text/javascript">
	$( "#select-demo" ).pwn_select();
</script>

Checkbox

Pwn checkboxes replace normal <input type="checkbox"> elements inside your form. The Pwn JavaScript library is required for this form element. For full syntax and options, see the JavaScript Components page. 

<label for="checkbox-demo" class="checkbox">
	<input type="checkbox" id="checkbox-demo"> Yes, I'd like to check out this free download
</label>

<script type="text/javascript">
	$( "label.checkbox input" ).pwn_checkbox();
</script>

Radio Button

Pwn radio buttons replace normal <input type="radio"> elements inside your form. The Pwn JavaScript library is required for this form element. For full syntax and options, see the JavaScript Components page. 

<label class="radio"><input type="radio" name="radio-demo" value="hero"> Die a hero</label>
<label class="radio"><input type="radio" name="radio-demo" value="villain"> Live long enough…</label>

<script type="text/javascript">
	$( "label.radio input" ).pwn_radio();
</script>

Buttons

Buttons are designed to be placed in many different locations throughout your site, not just inside forms. Thus, buttons are not required to be inside <form> tags to style properly.

All buttons have the btn class. You can add secondary classes to style the button for your situation. Buttons can be a, div, button, input type="submit", or span.
